# n8nProject

#To run the project, you need to have Node.js and npm installed on your machine. Then, follow these steps:
1. Clone the repository:
2. Navigate to the project directory:
3. Start the development server: docker compose -f compose.yml up -d 
4. Open your browser and go to http://localhost:5678 to see the application running.
5. You can now start building your n8n workflows and automations using the application. For more information on how to use n8n, please refer to the official documentation at https://docs.n8n.io/.
6. To stop the development server, run: docker compose -f compose.yml down
7. If you want to remove the containers and volumes, run: docker compose -f compose.yml down -v
8. For any issues or contributions, please open an issue or submit a pull request on the GitHub repository. Happy automating!
